# Market Sessions & HTF Levels

## Overview

The Market Sessions & HTF Levels indicator displays major market sessions and higher-timeframe reference levels directly on the chart.

It is designed for traders who use session structure, opens, and period boundaries as part of their chart workflow.

## What It Does

The script can display:

- major trading sessions
- daily, weekly, monthly, or quarterly open levels
- higher-timeframe reference lines
- optional historical levels
- period break markers

The exact feature set depends on the current script version and settings.

## Typical Use Cases

This type of tool can be useful for:

- intraday session mapping
- tracking higher-timeframe opens
- identifying context around key market periods
- improving chart organization

## Key Features

- Session overlays
- HTF open levels
- Historical level display options
- Intraday workflow support
- Visual chart references

## Main Inputs

Depending on the script version, common inputs may include:

- session visibility toggles
- open level visibility toggles
- previous period display
- line extension settings
- intraday-only filters
- label and color settings

## Interpretation Notes

This is a chart utility tool. It does not generate directional trade signals by itself.

It is intended to support traders who want clearer visual context for session-based and period-based analysis.
